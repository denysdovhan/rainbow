Structure

Кілька простих кроків, щоб почати використовувати Rainbow

Global

Doctype

Ми радимо вам використовувати HTML5 Doctype, адже Rainbow використовує найсвіжіші напрацювання в галузі веб-розробки для коректної роботи яких необхідний HTML5 Doctype.

Після підключення Rainbow шаблон вашої сторінки має вигляжати так:

<!DOCTYPE html>
<html>
  <head>
    <title>Rainbow Template</title>
    <link rel="stylesheet" type="text/css" href="css/rainbow.css">
  </head>
  <body>
    <h1>Hello, world!</h1>
  </body>
</html>

Reset CSS

Ми використовуємо модифікований Normalize.css, a project by Nicolas Gallagher and Jonathan Neal

Colors

Rainbow підтримує 15 додаткових кольрів, але поки що не всі елементи підтримують всю палітру. В документації до кожного елемента ми будемо присвоювати значки з кольорами, які підтримує цей елемет. Зараз доступні такі кольори: Red, Orange, Yellow, Lime, Green, Lightblue, Cyan, Blue, Darkblue,Purple, Pink, Olive, Brown, Maroon, Black
