Яскравий

Rainbow створено для того, щоб легко створювати яскравий дизайн. Він підтримує 16 кольорів та 2 типи дизайну (flat & volume)

Сучасний

В Rainbow використані найсучасніші технології. Він підтримує сучасні стандарти веб-розробки HTML5 та CSS3. Написаний на LESS та розміщений на GitHub.

Download

Щоб почати працювати з Rainbow скачайте архів з фреймворком, або підключіть фреймворк через HTTP. Якщо ви хочете швидко почати роботу, ми радимо скачати вам готову скомпільовану та мініфіковану версію.

Отримайте по HTTP

Ви можете динамічно підключити скомпільований та мініфікований файл стилів протоколом HTTP. Вставте цю строку в тег <head>

Імпортуйте в LESS

Віддалено імпортуйте Rainbow в свій LESS-файл та розширюйте можливості фреймворка використовуючи команду @import

What`s included?

Rainbow ввібрав найсучасніші ідеї та тенденції з дизайну та веб-розробки.

Structure

Базові стилі для body, обнулення стандартної типографії, grid system, базові кольорові теми для посилань

Typography

Стилы для основних типографічних елементів, таблиць, блоків коду, форм та кнопок.

Navigations

Стилі для навігації, табів, пілсів, сайдбару, навігаційного бару, хлібних крошок та пагінації.

Media

Media components: alerts, media-grid, media-lists, banners, progress-bars.
