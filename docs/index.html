We presents Rainbow

Cross-browser

You can not worry for support of styles in outdated browsers - Rainbow looked after already about it.

Colorful

Rainbow was created to make it easy to create a bright design. It supports 16 colors and 2 types of design (flat & volume)

Modern

In Rainbow utilized the latest technologies. It supports the modern standards of web development HTML5 and CSS3. Written in LESS and based on GitHub.

Download

For start working with Rainbow you have to download the archive with framework or connect framework via HTTP. If you want to get started quickly, we advise you to download compiled and ready minified version.

What`s included?

Rainbow incorporates the latest ideas and trends in design and web development.

Structure

Basic styles for the body, reset styles, grid system, basic color themes for links.

Typography

Styles for basic typographic elements, tables, code blocks, forms and buttons.

Navigations

Styles for navigation tabs, pills, sidebar, navigation bar, pagination and breadcrumbs.

Media

Media components: alerts, media-grid, media-lists, banners, progress-bars.
